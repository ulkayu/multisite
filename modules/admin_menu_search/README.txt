INTRODUCTION
------------

Admin Menu Search intends to help admin to search through list of menus
available in Drupal Toolbar providing autocomplete list. Providing a fast
access to all administration links for keyboard lovers.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/admin_menu_search

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/search/admin_menu_search


REQUIREMENTS
------------

Enable Admin Toolbar Extra Tools module for a better experience (optional).


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.
   See: https://www.drupal.org/node/895232 for further information.


CONFIGURATION
-------------

No configuration is needed.


KEYBOARD SHORTCUT
-----------------

 * Alt + M - Focus admin toolbar Menu Search.
